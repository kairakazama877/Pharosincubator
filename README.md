# Pharos Builder Incubator Program

Full-stack web application for the Pharos Builder Incubator Program.
Includes landing page, API backend, and documentation.
Ready to import into GitHub/GitLab.

## Stack
- Frontend: Next.js (React)
- Backend: Node.js + Express
- Database: PostgreSQL (via Prisma - optional)
- Auth: JWT (ready)

## Getting Started
```bash
cd frontend
npm install
npm run dev

cd ../backend
npm install
npm run dev
```
